[Evidence Labs](https://labs.evidence.dev) contains a collection of experimental Evidence components which can be imported into your project.

## Available Components
- Cohort Analysis
- Footnote
- Github Stars Counter
- Google Sheet Embed
- Grid
- Leaflet
- Maps
- PDF Embed
- Slider
- Summarizer
